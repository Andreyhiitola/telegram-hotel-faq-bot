# Telegram FAQ Бот

## Быстрый старт

1. Получите токен от @BotFather в Telegram
2. Отредактируйте файл .env и вставьте токен
3. Запустите: docker-compose up -d --build

## Команды управления

- Запуск: docker-compose up -d --build
- Остановка: docker-compose down
- Логи: docker-compose logs -f
- Перезапуск: docker-compose restart

## Редактирование FAQ

Откройте bot.py и измените словарь FAQ под свои нужды.
